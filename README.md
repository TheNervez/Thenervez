# Hi, I'm Adeola 👋

I am a Biomedical Engineering & Applied Physics researcher focused on **perception systems, optical sensor architectures, and wearable health monitoring hardware**. I build automated pipelines to process multi-channel sensor traffic, characterize signal-to-noise anomalies, and turn raw time-series data into actionable hardware optimizations.

## 🔬 Core Focus Areas
* **Perception & Sensor Fusion:** Integrating and calibrating asynchronous sensor streams (IMUs, biosensors) for real-world environmental understanding.
* **Optical & Nanophotonic Systems:** Developing parametric simulation pipelines to optimize photon capture efficiency and waveguide geometries.
* **Wearable Health Tech:** Designing hardware-level logging pipelines and custom multi-channel arrays for real-time diagnostic applications.

## 🛠️ Technical Stack
* **Languages:** Python (NumPy, Matplotlib, Automated Pipelines), C++, MATLAB
* **Environments & Tools:** Linux (Shell, Log Debugging), Git/GitHub, SPICE, Verilog, Arduino/Embedded Interfacing

## 🚀 Current Engineering Work
* Simulating automated parametric sweeps for nanophotonic waveguide optimization to drive photon capture efficiency.
* Characterizing signal degradation and sensor displacement artifacts in wearable multi-channel arrays.
* Developing Temporal Convolutional Network (TCN) and Graph Neural Network (GNN) tools for real-time data processing and circuit validation.
